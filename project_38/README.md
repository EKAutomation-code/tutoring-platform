
# BTC Price Tracker Bot 

Telegram-бот для отслеживания цены Биткоина. Бот следит за ценой BTC и присылает уведомление, когда она достигает заданной пользователем цели.

## Возможности

- Отслеживание текущей цены BTC
- Установка целевой цены (бот сам пришлёт уведомление)
- Автоматическое обновление цели
- Остановка отслеживания
- Безопасное хранение токена через `.env`

## Команды бота

```
`/start`    | Приветствие и инструкция 
`/price`    | Показать текущую цену BTC 
`/mytarget` | Показать текущую цель 
`/stop`     | Остановить отслеживание 
`/help`     | Справка по командам 
```


## Как это работает

1. Пользователь отправляет боту число — желаемую цену в долларах
2. Бот начинает мониторинг цены BTC (проверка каждые 15 секунд)
3. Когда цена достигает цели или превышает её — бот присылает уведомление

## Технологии

- Python 3.x
- `python-telegram-bot` — работа с Telegram API
- `requests` — HTTP-запросы к Binance API
- `python-dotenv` — работа с переменными окружения

## Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/EKAutomation-code/work.git
cd work/project_38
```

### 2. Создайте виртуальное окружение

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate     # Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Создайте файл .env

Скопируйте .env.example в .env и добавьте свой токен:

```env
BOT_TOKEN=ваш_токен_от_botfather
```

### 5. Запустите бота

Скопируйте .env.example в .env и добавьте свой токен:

```bash
python project_38.py
```

## Получение токена бота

- Найдите в Telegram @BotFather
- Отправьте команду /newbot
- Придумайте имя и username бота
- Скопируйте полученный токен в файл .env
