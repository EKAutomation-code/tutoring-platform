# Парсер книг с scrapeme.live

Скрипт для сбора данных об товарах с интернет-магазина `scrapeme.live` (демо-сайт для тренировки парсинга). Автоматически обходит все доступные страницы каталога, извлекает названия, цены и ссылки на товары, сохраняет результат в CSV-файл.

## Структура проекта

```
project_31/
├── project_31.py # Скрипт парсера
├── shop_products.csv # Результат парсинга (таблица с данными)
└── README.md # Описание проекта
```

## Что делает скрипт

- Автоматически обходит **все страницы** каталога (останавливается, когда товары заканчиваются)
- Для каждого товара извлекает:
  - Название (`title`)
  - Цену (`price`)
  - Ссылку на товар (`link`)
- Обрабатывает ошибки сети и отсутствие данных
- Добавляет задержку между запросами (вежливый парсинг)
- Сохраняет результат в файл `shop_products.csv`

## Технологии

- Python 3.x
- `requests` — HTTP-запросы с кастомными заголовками
- `BeautifulSoup` (bs4) — парсинг HTML
- `pandas` — работа с данными и сохранение в CSV
- `time` — задержка между запросами

## Ключевые особенности

- **Автоматическое определение конца каталога** — скрипт сам останавливается, когда товары заканчиваются
- **Обработка ошибок** — защита от проблем с сетью и отсутствующих тегов
- **User-Agent** — имитация реального браузера
- **Timeout** — ограничение времени ожидания ответа

## Настройка

Чтобы изменить задержку между запросами, отредактируйте строку в project_31.py:
### Было: 1 секунда

```python
time.sleep(1)
```

### Стало: 2 секунды
```python
time.sleep(2)
```
