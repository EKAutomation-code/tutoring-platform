# Парсер книг с books.toscrape.com

Простой скрипт для парсинга данных о книгах с тестового сайта-песочницы `books.toscrape.com`. Извлекает названия, цены и рейтинги книг, сохраняет результат в CSV-файл.

## Структура проекта

```
project_27/
├── project_27.py # Скрипт парсера
├── books.csv # Результат парсинга (таблица с данными)
└── README.md # Описание проекта
```

## Что делает скрипт

- Обходит первые 5 страниц каталога
- Для каждой книги извлекает:
  - Название (`title`)
  - Цену (`price`)
  - Рейтинг (`rating` — от `One` до `Five`)
- Формирует таблицу данных с помощью pandas
- Сохраняет результат в файл `books.csv`

## Технологии

- Python 3.x
- `requests` — HTTP-запросы
- `BeautifulSoup` (bs4) — парсинг HTML
- `pandas` — работа с данными и сохранение в CSV
- `time` — вежливая задержка между запросами
